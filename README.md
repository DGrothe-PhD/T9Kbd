# T9Kbd

T9 keyboard app for Windows.

Type by clicking on the buttons or using the number keys of your keyboard. You can enter a full text with just one finger.

## Shortcuts

|Shortcut   | Function            |
|-----------|---------------------|
| `Return`  | Linebreak           |
| `F3`      | Copy to clipboard   |
|`Backspace`| Delete last letter  |
| `F6`      | Clear text field    |
